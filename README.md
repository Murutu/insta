## Insta App

> This is a NextJS App that allows users to showcase their pictures to other users.

​

![insta](public/images/insta.jpg)

​

## Technologies Used

- React
- Tailwind Css
- Firebase
- Next Auth

​

## Features

- Google Authentication
- SignOut
- Post pictures
- Like pictures
- Comment on pictures

​

## Deployment

Deployed to [Vercel](https://vercel.com/)
